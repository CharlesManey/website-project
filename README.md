# website-project



## Description
First, I went into my terminal in VS Code and navigated to the proper directory, 2026-RTT-27, using the cd command. I then used the mkdir command to create a new folder called website-project and navigated into it. After creating the new folder, I initialized it with the command git init. I right-clicked on the website-project folder inside the VS Code explorer and clicked "New File." I proceeded to create a new file for my index.html and my styles.css. I started with a basic HTML boilerplate, edited my title, and added a link to my CSS. Then, using the commands git add . and git commit -m '', I created my first commit.Next, I went to GitHub and created a new repository for my website-project. In order to link my project folder and repository, I pasted the three necessary lines from GitHub into my terminal. Now that my project folder and repository are linked, I went back to my terminal and executed my first push using the command git push origin main. I had a landing page in mind, so it was time to add some sections to my HTML; I added a nav, hero, and footer section. I then used a very descriptive prompt in ChatGPT to generate an idea and the background plus hero image I would be using. I then asked ChatGPT to give me a step-by-step process on how to bring this idea to life. I hand-typed everything and learned a lot about CSS along the way. I had to make multiple adjustments to make it fit on one page, and thanks to Live Server, I could see everything in real time!
